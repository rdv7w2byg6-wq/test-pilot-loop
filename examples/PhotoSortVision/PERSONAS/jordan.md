# Jordan — Everyday User

## Profile
- **Age:** 34
- **Tech comfort:** Uses Mac daily for email, browsing, photos
- **Relevant experience:** Uses Apple Photos occasionally, has organized files manually before
- **Goal:** Clean up a messy photo folder without losing anything

## Behavioral Rules
- Reads labels before tapping — but skips anything longer than one sentence
- Expects standard macOS patterns (toolbar, sidebar, ⌘Z for undo)
- Will not right-click unless stuck
- Abandons a flow if confused for more than 10 seconds
- Trusts "destructive" actions less — needs confirmation before moves or deletes

## What Jordan Catches
- Confusing labels and icons
- Missing confirmation dialogs before file operations
- Flows that require technical knowledge (file paths, terminal commands)
- Inconsistent navigation patterns
